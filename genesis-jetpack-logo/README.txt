== Description ==

A simple plugin to allow you to use the Jetpack Logo feature in your Genesis Child Theme.
You have a few options to set your logo demensions so play around with them. You may need
to regenerate thumbnails if the logo was uploaded before activating this plugin.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `genesis-jetpack-logo.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Set and select your logo size in the options area located under the Genesis main menu.
4. Go to apperance > customize and let your logo.

== Frequently Asked Questions ==

= Is this plugin compatiable with non Genesis themes? =

No, this plugin is Genesis specific.
